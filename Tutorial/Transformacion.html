Transform

Con HTML5 y CSS3 podemos utilizar la propiedad 'transform' para modificar características de un < div > como la posición (position), la rotación (rotation), la escala (scale) o el -menos utilizado- el sesgo (skew). Este cambio de posición, tamaño o rotación se produce al instante (no hay animación en este cambio, para ello habría que utilizar 'transition' o 'animation').
