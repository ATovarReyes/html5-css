IMAGENES

Imagenes

las imágenes para web son recursos gráficos digitales (fotografías, ilustraciones, iconos) esenciales para la comunicación visual, el diseño y la experiencia de usuario (UX) en sitios web.

optimizadas en tamaño y peso

Deben estar optimizadas en tamaño y peso para no afectar la velocidad de carga, siendo fundamentales para transmitir emociones, mejorar la estética y transmitir mensajes rápidamente

Optimización y Formatos:

Optimización y Formatos:

JPG: Alta compresión, ideal para fotografías.
PNG: Admite fondos transparentes, ideal para logotipos e infografías.
SVG: Mejor formato para iconos y gráficos vectoriales por su escalabilidad y bajo peso.
WebP/AVIF: Formatos modernos que ofrecen alta compresión y calidad superior a JPG/PNG.

El peso ideal para imágenes web

El peso ideal para imágenes web es inferior a 200-300 KB para asegurar una carga rápida. Las dimensiones recomendadas son un máximo de 1200-2048 px de ancho para imágenes grandes y 72 dpi (resolución web). El formato recomendado es JPG para fotos y WebP para máxima optimización, comprimiendo entre 60-80% de calidad

Dimensiones Recomendadas (Píxeles)

Imágenes de fondo/portada: 1920 x 1080 px o hasta 2560 x 1400 px.
Imágenes de contenido/blog: 1200 x 800 px o similar.
Banners: 1200 x 400 px (largo) o 300x250 px (medio).
Resolución: 72 ppp (puntos por pulgada).
Recomendaciones de Peso y Formato
Peso Ideal: Menos de 200 KB por imagen.
Formato de Archivo: WebP (mejor compresión), JPG (buena calidad/peso), PNG (para transparencia o iconos).
Herramientas de Compresión: TinyPNG, Optimizilla
